# Node.js

> Node.js 是运行在服务端的 JavaScript。
>
> Node.js 是一个基于Chrome JavaScript 运行时建立的一个平台。
>
> Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行Javascript的速度非常快，性能非常好。

查看当前 Node 版本

```javascript
//查看当前 Node 版本
$ node -v
v4.4.3
```

交互模式

```javascript
$ node
> console.log('Hello World!');
Hello World!
```
