# Notes

1. Delete loop edge / points:

   > Slide loop edge -> offset 100% -> Melt -> Optimize Points

2. Sharp edge:

   > Select  edge -> Bevel -> Subdivision 1

3. Delete extra points:

   > Object —> Current State to Object
